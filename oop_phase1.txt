Q8. Explain working of self keyword in Python.

The self keyword in Python refers to the current object of a class. It is used to access instance variables and methods inside the class.

When a method is called using an object, the object is automatically passed as the first argument to the method. This first parameter is written as self by convention.

self helps to differentiate between instance variables (belonging to object) and local variables (temporary variables inside method).

 Explanation

self refers to the object that calls the method.
In self.name = name, the value is stored inside the object.
When s1.display() is executed, self refers to s1.
Using self, object data can be accessed inside methods.

🔹 Important Points

self is not a keyword, but a convention.

It must be written as the first parameter in instance methods.

It is automatically passed when method is called using object.

It is used to access and modify object data.

The __init__() method in Python is a special method known as a constructor. It is automatically executed when an object of a class is created. 
Its main purpose is to initialize the data members (instance variables) of the class.
